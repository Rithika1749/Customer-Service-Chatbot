
# Customer Service Chatbot using Python and NLP

## Overview

This project is a simple Customer Service Chatbot built using Python and Natural Language Processing (NLP).

The chatbot can answer common customer queries such as:
- Business Hours
- Contact Information
- Refund Policies
- Payment Methods
- Delivery Information
- Order Tracking

## Features

- NLP-based preprocessing
- Tokenization
- Stopword Removal
- Intent Recognition
- Conversation Logging
- Query Analytics

## Technologies Used

- Python
- NLTK
- Google Colab

## Future Improvements

- Flask Web Interface
- AI-powered Responses
- Voice Assistant Integration
